Apple HomeKit Doorbell

Expose a virtual doorbell accessory so Apple Home and HomePod receive ring notifications.

How to set up:

  1. Enable the toggle above and save configuration.
  2. Restart the NukiBlinker service (HomeKit starts at boot).
  3. Open the Apple Home app on your iPhone/iPad.
  4. Tap + → Add Accessory.
  5. Choose "More options..." — the NukiBlinker Doorbell should appear.
  6. Enter the setup code shown below when prompted.

This is the PIN you enter in Apple Home when pairing. Leave empty to auto-generate one on first start.

Directory where pairing state is stored. Keeps pairings across restarts.

Docker note: HomeKit uses mDNS/Bonjour for discovery. If running in Docker, you must use network_mode: host in your docker-compose.yml, otherwise Apple Home won't find the accessory.
